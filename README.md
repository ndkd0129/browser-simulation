# Browser Simulation Project

This project simulates a search flow on a mock Google homepage and navigates to a premium landing page.

## How to View

### Local
Simply open `index.html` in your web browser.

### Deployment (GitHub Pages)

This project is configured to automatically deploy to GitHub Pages using GitHub Actions.

1.  Push this code to a GitHub repository.
2.  Go to the repository **Settings** > **Pages**.
3.  Under **Build and deployment**, ensure **Source** is set to **GitHub Actions**.
4.  The site will be deployed automatically on every push to the `main` or `master` branch.


## Project Structure

- `index.html`: The self-driving Google homepage mock.
- `landing.html`: The premium "Memory Space" landing page.
